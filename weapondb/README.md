# CS2 Item Database
База даних, яка містить в собі інформацію про предмети CS2. База даних передбачена під створення маркетплейсу предметів по типу csfloat.com, cs.money, dmarket.net, та інших; але може бути використана і для інших цілей — наприклад, менеджмент гравців для моду CS:GO по типу ClassicCounter чи CS Supremacy. База даних зроблена Бойко Денисом.
## Технічна інформація
Frontend бази даних зроблений на VCL (C++) через RAD Studio, Backend бази даних зроблений на PostgreSQL 18 через pgAdmin 4. Комунікація між ними зроблена через бібліотеку libpq.dll та FireDAC. 
## Інструкції з налаштування
1. Клонуйте репозиторій.
2. Створіть нову базу даних в pgAdmin 4.
3. Використовуючи Query Tool, запустіть скрипт src\SQL\final_schema.
4. Використовуючи Query Tool, запустіть скрипт src\SQL\add_start_data.
5. Встановіть налаштування вашої бази даних PostgreSQL у src\CPP\Win32\Debug\WeaponDB_v2.ini
6. Відкрийте проект src\CPP\WeaponDB_v2.cbproj у RAD Studio.
7. Скомпілюйте проект.
## Запуск додатку
Після компіляції проекту, .exe буде знайдено у src\CPP\Win32\Debug. Для правильної роботи проекту потрібно, щоб був відкритий pgAdmin 4 з належною базою даних. Також, потрібні декілька .dll файлів для правильної роботи комунікації між frontend та backend — їх надано разом з source кодом у src\Win32\Debug.
